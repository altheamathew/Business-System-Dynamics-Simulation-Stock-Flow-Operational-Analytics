📊 Business System Dynamics Simulation
📘Overview

This university project focused on solving a company’s stock-flow and operational management problems using **System Dynamics Modelling** and **iThink simulation software**.

The project analysed how:

- Sales demand
- Production capacity
- Delivery delays
- Staffing
- Revenue
interact inside a business system over time.
The goal was to identify operational bottlenecks and simulate strategies to improve business performance.



🎯 Use Cases / Key Variables
Key Variables Analysed

- Order Backlog
- Order Fill Rate
- Production Capacity
- Sales Representatives
- Delivery Delay
- Marketing Budget
- Revenue


🚀 Motivation

The company faced issues with:

- Increasing customer orders
- Delivery delays
- Production pressure
- Growing backlog
- Resource allocation inefficiencies

This project aimed to answer:

- What causes operational instability?
- Why do delivery delays increase?
- How does staffing affect revenue growth?
- What happens when demand exceeds capacity?

The simulation model helped predict business behaviour before implementing real-world changes.


🛠️ Technical Implementation

Techniques Used

- System Dynamics Modelling
- Causal Loop Diagrams (CLD)
- Stock & Flow Modelling
- Feedback Loop Analysis
- Information Smoothing
- Non-linear System Simulation

Software Used

- iThink Simulation Software


📈 Results

The simulation revealed that:

✅ Higher sales increased revenue rapidly

✅ Production capacity became the main bottleneck

✅ Backlog increased when demand exceeded capacity

✅ Delivery delays negatively impacted future customer orders

✅ Hiring delays slowed operational recovery

### Major Finding

Sales growth without capacity expansion creates operational instability.



💡 Insights

Key Business Insights

- Capacity planning is critical for sustainable growth
- Faster order fulfilment improves customer retention
- Delivery delays directly reduce future demand
- Operational delays create long-term system instability
- Data-driven simulation improves strategic decision-making
